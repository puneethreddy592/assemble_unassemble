SimpleRISC Architecture

Complete documentation for the SimpleRISC architecture. Use this reference to understand the instruction set, encoding formats, and more.

1. Instruction Format

Each instruction must follow the opcode format as defined in the assembler.

Supported Instructions

Arithmetic & Logical
add, sub, mul, div, mod, and, or, lsl, lsr, asr
Memory Operations
ld, st
Control Flow
b, beq, bgt, call, ret
Other
cmp, not, mov, nop, hlt

2. Instruction Size

Every instruction occupies 4 bytes in memory.

3. Comment Syntax

Comments start with a ; and extend to the end of the line.

Example:

add r1, r2, r3 ; This is a comment

4. Label Guidelines

Label Definition

Labels start with a . and end with a :

Example:

.loop: add r1, r2, r3 b .loop

Predefined Labels

.start, .main, and .org define the starting address.

Format:

.start 0x1000

If .start is provided, all addresses will be shifted accordingly.

Branch Labels

Labels used in b, beq, bgt, call must be predefined in the assembly file.

Example:

b .loop

5. Register Usage

Register Naming

Registers are named r0 to r15.

Example:

add r1, r2, r3

Immediate vs Register Operand

Some instructions support both immediate values and registers.

Immediate values can be decimal, binary (0b...), or hex (0x...).

Example:

mov r1, 10 ; Immediate value add r2, r1, r3 ; Register operands

6. Instruction Variants

Modified Arithmetic Instructions (addu, addh, etc.)

  • u ‚Üí Unsigned mode
  • h ‚Üí High mode

Example:

addu r1, r2, 10 ; Unsigned addition addh r3, r4, r5 ; High mode addition

Load/Store Format

Load and Store instructions have special syntax:

ld r1, 0x1000 [r2] ; Load from memory address st r3, 0x2000 [r4] ; Store to memory address

7. Memory Addressing & Constants

Allowed Immediate Value Ranges

  • Arithmetic Immediate (16-bit signed): -32,768 to 32,767
  • Branch Offset (27-bit signed, divided by 4): -67108864 to 67108863

Memory Access Guidelines

Memory addresses must be properly aligned (multiples of 4).

Example:

ld r1, 0x1004[r2] ; Valid

8. Control Flow Guidelines

Branching & Jumping

Use b, beq, bgt, and call with labels.

Example:

beq r1, .loop

Function Return

Use call to jump and ret to return.

Example:

call .function ret

Halting Execution

The hlt instruction stops the program.

Example:

hlt

9. Example Assembly Program

.start 0x1000 .main: mov r1, 10 mov r2, 20 add r3, r1, r2 b .end .loop: sub r1, r1, 1 bgt r1, .loop .end: hlt
